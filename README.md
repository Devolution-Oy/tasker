# Devolution tasker

> A GitHub App built with [Probot](https://github.com/probot/probot)

## Briefly
Tasker App handles Devolution Oy work tasks and new issue creation.

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```
